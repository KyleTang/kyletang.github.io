仓颉语言库 API

std.math 包

功能介绍

math 包提供常见的数学运算，常数定义，浮点数处理等功能。

包括了以下能力：

  1. 科学常数与类型常数定义；
  2. 浮点数的判断，规整；
  3. 常用的位运算；
  4. 通用的数学函数，如绝对值，三角函数，指数，对数计算；
  5. 最大公约数与最小公倍数。
