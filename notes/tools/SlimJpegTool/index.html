          # 瘦图

          # 1. 说明

          # 1.1 简介

          图片瘦身工具，可在维持原画质和分辨率的情况下对图片批量压缩处理，极大的节省磁盘空间，并行处理速度很快，界面简单。

          大部分场景可以减少60%至80%的空间占用，少部分场景可以减少高达85%至90%，手机照片在质量选项85的情况下，也能节省50%至60%的空间占用。

          将jpg,png,heic格式文件通过mozjpeg算法压缩，在默认质量选项75进行压缩处理，压缩后通过100%缩放1:1对比几乎无明显差别。

          # 1.3 推荐压缩质量选项

          需求 场景举例 质量选项建议 可节省空间
          高保真需求 手机照片、高清资料图 一般75、85 节省60%~70%
          低保真需求 屏幕截图、视频截图、一般资料 40、50即可 节省70%~85%
          无保真需求 能看出是啥就行 10 高达90%以上
          无脑选项 满足几乎所有场景 默认75 60%~80%

          # 3. 特性

          1. 可设定压缩质量，可选择可输入，可自行压缩对比，根据图片用途，选择适合的质量选项
          2. 批处理处理多个目录，包含子目录；可拖拽目录、粘贴路径。
          3. 运行模式选择：原目录处理或输出到新目录
          4. 支持并行处理执行，充分利用多核资源
          5. 支持jpg,png,heic格式的待处理文件
          6. 小于300KB的文件不处理（可自定义限制值），如果处理后大于原文件，也不处理。
          7. 可自动修复图片文件的扩展名错误，目前支持gif,jpg,png,heic,webp.

          # 6. 温馨提示

          • 质量选择：建议多次压缩对比，根据自己场景选择合适的质量选项；
          • 并行处理：如果是普通硬盘，建议不要开启全部线程，可能硬盘忙不过来；固态硬盘可以选择全部线程；
          • 效果不佳：色彩简单的PNG比jpg的压缩效果更好，和图片内容有关，压缩后的确会出现压缩后体积变大，这个程序会跳过压缩，这类场景的原图通常不足1MB。
          • 扩展名：
            • png压缩成jpg，扩展名还是png，不影响使用，可防止引用出错。
            • heic压缩成jpg格式，扩展名会变成jpg
