                    # 更新说明

                    # 1.0.7.3 - 2024.05.07

                    • [修复] heic转jpg失败导致的闪退问题

                    # 1.0.7.2 - 2024.03.20

                    • [优化] 支持按文件大小范围过滤待处理图片
                    • [优化] heic转换jpg初次处理，默认画质从70%调整成90%
                    • [新增] 支持heic转jpg增加_IOS尾缀
                    • [新增] 支持保留文件修改日期
                    • [修复] 当原路径为根路径时，无法创建目的目录的bug

                    # 1.0.7 - 2024.01.10

                    • [优化] 代码结构
                    • [调整] 修复bug、判断jpg文件头

                    # 1.0.6 - 2022.05.28

                    • [优化] 解决一些小bug，提高稳定性，在处理扩展名为heic，但非heic格式时dll调用会崩溃等。
                    • [优化] 支持拖拽文件夹
                    • [优化] 支持自动修复文件扩展名错误
                    • [优化] 增加主日志

                    # 1.0.5 - 2022.05.26

                    • [新增] 增加路径时，支持选择文件夹
                    • [新增] 检查版本更新
                    • [新增] 自定义小文件限制
                    • [优化] 修复小bug，优化代码，完善统计信息和日志，增加路径判重处理

                    # 1.0.4 - 2022.05.26

                    • [调整] 支持选择运行模式，原目录处理还是输出到新目录
                    • [新增] 支持设置输出目录
                    • [新增] 支持日志输出
                    • [优化] 强壮性增强，检查用户输入的路径字符合法性

                    # 1.0.3 - 2022.05.25

                    • [新增] 支持heic格式
                    • [新增] 支持过滤文件类型 jpg,png,heic
                    • [修复] 修复多线程无法停止的bug
                    • [新增] logo
                    • [修复] 修复崩溃：新增heic2jpg.exe文件，因heic多线程调用会崩溃，改用多进程的方式调用，将dll封装到heic2jpg.exe中调用。
                    • [优化] 忽略文件小于300KB
                    • [优化] 当处理后的文件大于原文件，则删除处理后的文件。
                    • [新增] 处理完成后显示处理的统计信息。
                    • [调整] 不再使用libs目录，将里边的文件放到与主程序同级目录，cjpeg.exe移到上一级目录

                    # 1.0.2 - 2022.05.24

                    • [新增] 支持多线程执行

                    # 1.0.0 - 2022.05.23

                    • [新增] 可指定压缩质量
                    • [新增] 批处理处理多个目录，包含子目录
                    • [新增] 覆盖原文件，并自动备份，保留创建时间
