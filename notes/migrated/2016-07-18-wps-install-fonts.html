# wps提示缺失字体

2016.07.18

安装wps，提示系统缺失字体

从这里下载字体：http://download.csdn.net/download/wl1524520/6333049 （本地下载地址： wps_symbol_fonts ）

文件名为 wps_symbol_fonts.zip 1.解压缩

sudo mkdir -p /usr/share/fonts/wps_symbol_fonts
sudo chmod 755 /usr/share/fonts/wps_symbol_fonts
sudo unzip wps_symbol_fonts.zip -d /usr/share/fonts/wps_symbol_fonts

2.生成缓存配置信息

cd /usr/share/fonts/wps_symbol_fonts
sudo mkfontdir
sudo mkfontscale
sudo fc-cache
