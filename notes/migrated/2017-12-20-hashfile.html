      # 计算文件的MD5、SHA1、SHA256值

      2017.12.20

      # 1. windows

      自带工具 certutil 或 第三方程序 HashCalc(http://www.slavasoft.com/hashcalc/)

      certutil -hashfile d:\file.iso SHA1

      d:\file.iso 文件名

      SHA1 是算法，算法支持：MD2 MD4 MD5 SHA1 SHA256 SHA384 SHA512

      命令帮助：

      certutil /?

      # 2. Linux

      #计算MD5
md5sum /opt/file.iso
#计算sha1
sha1sum /opt/file.iso
#计算SHA256
sha256sum  /opt/file.iso
